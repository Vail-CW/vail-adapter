# Hardware

Board designs and printable cases for the Vail Adapter, organized by variant.
Each variant folder holds the PCB design and, where there is one, the matching
3D printable case.

- `basic-adapter/`
  - `pcb/` - KiCad project and gerbers for the Basic PCB
  - `case/` - printable case (STL) and Fusion 360 source (.f3d)
- `advanced-adapter/`
  - `pcb/` - KiCad project and gerbers for the Advanced PCB (adds radio output)
  - `case/` - printable case and Fusion 360 source
- `button-hat/`
  - `pcb/` - KiCad project and gerbers for the button hat add on

Order bare PCBs from JLCPCB or PCBWAY using the gerber zip in each `pcb/` folder,
or get a kit or a finished unit from [shop.ke9bos.com](https://shop.ke9bos.com).
Step by step assembly guides are at [vailadapter.com/assembly](https://vailadapter.com/assembly).

# VAIL SUMMIT v0.30 Release Notes

**Release Date:** 2026-01-18

## What's New

### Smoother Audio Playback
All training modes now play morse code without freezing the screen. You can see the UI update while audio plays, and pressing ESC immediately stops playback.

**Affected modes:**
- Hear It Type It
- CW Academy (Copy, Send, and QSO Practice)
- Koch Method

### Quick Volume Access
Press **V** from the main menu to quickly adjust volume.

## Bug Fixes

### Hear It Type It Settings
Fixed an issue where navigating the settings screen required pressing DOWN twice to reach the Start button. Settings now load correctly when entering the mode.

## Testing Notes

Please verify:
1. Audio plays smoothly without glitches in all training modes
2. Screen updates during morse playback (not frozen)
3. ESC key stops audio immediately
4. Saved settings (mode, WPM, group length) load correctly in Hear It Type It
5. V key opens volume settings from main menu

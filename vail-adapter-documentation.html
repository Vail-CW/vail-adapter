Overview

The Vail Adapter is a USB-connected Morse code key/paddle interface that allows you to use physical Morse code keys with computer-based CW training software and online platforms. Designed primarily for use with Vail and VBand, it provides low-latency keying with optional built-in keyer functionality and sidetone generation.

The adapter transforms your traditional Morse code paddles or straight keys into a modern USB HID device, enabling seamless integration with digital platforms while maintaining the authentic feel of mechanical keying.

Key Features

Universal Compatibility

Works with Vail, VBand, and other Morse code software. Functions as both keyboard and MIDI device.

Built-in Keyer

Implements all nine keyer modes from Vail directly in hardware for zero-latency operation up to 50+ WPM.

Sidetone Generator

Optional built-in audio feedback helps with timing and latency compensation.

Focus Independence

Continue keying even when the software window loses focus - perfect for multi-tasking.

Capacitive Touch

Advanced models support capacitive touch keying - no physical contact required.

Radio Integration

Advanced PCB version includes radio keying outputs for direct transceiver control.

Installation & Setup

Easy Installation (Recommended)

  1. Acquire Hardware: Get a Seeeduino XIAO SAMD21 or compatible board
  2. Download Firmware: Get the latest UF2 file from the releases page
  3. Enter Bootloader Mode: Double-tap the reset button on your XIAO board
  4. Install Firmware: Copy the .uf2 file to the XIAO mass storage device
  5. Verify Installation: The device should reboot and appear as a USB HID device
The XIAO will appear as a removable drive when in bootloader mode. Simply drag and drop the firmware file!

Choosing the Right Firmware

Firmware files are named according to this pattern: vail-adapter_[CONFIG]_[BOARD].uf2

  • vail-adapter_BASIC_PCB_V2_xiao.uf2 - For XIAO with Basic PCB V2 (most common)
  • vail-adapter_ADVANCED_PCB_xiao.uf2 - For XIAO with Advanced PCB
  • vail-adapter_NO_PCB_GITHUB_SPECS_xiao.uf2 - For breadboard builds
  • vail-adapter_BASIC_PCB_V2_qtpy.uf2 - For QT Py boards

Advanced Installation (From Source)

For developers or those needing custom configurations:

  1. Install Arduino IDE
  2. Add required board packages
  3. Install MidiUSB and Keyboard libraries
  4. Modify config.h for your hardware
  5. Compile and upload the sketch

Wiring Guide

Basic Paddle Wiring

Basic Wiring Diagram Placeholder
Insert wiring diagram showing paddle connections

For most users, only three connections are needed:

  • Ground (GND): Connect to the common terminal of your paddle (usually center)
  • Dit Pin: Connect to the dit paddle (usually left side)
  • Dah Pin: Connect to the dah paddle (usually right side)

TRS (Headphone) Jack Wiring

For a standard 3.5mm TRS connector:

Tip (T) → Dit Pin Ring (R) → Dah Pin Sleeve (S) → Ground (GND)

Advanced Wiring Options

Sidetone Buzzer

Connect a piezo buzzer or small speaker between the piezo pin and ground for audio feedback.

Capacitive Touch Pads

Connect conductive pads or screws directly to the capacitive touch pins - no ground connection needed!

Radio Keying (Advanced PCB Only)

The Advanced PCB version provides direct radio keying outputs that can control your transceiver's CW keying line.

Audio Output (Advanced PCB Only)

The Advanced PCB includes a dedicated headphone/speaker jack that routes the sidetone audio output. When headphones or speakers are connected, the built-in buzzer is automatically bypassed, allowing for private listening or higher quality audio output.

Verify your radio's keying requirements before connecting. Some radios key with a low signal, others with high. Adjust the RADIO_KEYING_ACTIVE_LOW setting in firmware accordingly.

Updating Firmware

Web-Based Update Tool (Recommended)

The easiest way to update your Vail Adapter firmware is using the dedicated web tool:

Vail Web Updater: https://vail-cw.github.io/vail_web_updater/

This all-in-one tool allows you to:

  • Select your specific Vail Adapter model
  • Automatically put the device into bootloader/storage mode
  • Download the correct UF2 firmware file
  • Complete the entire update process in your browser

Manual Update Process

If you prefer the manual method, the Vail Adapter supports easy firmware updates without special tools:

  1. Download the latest firmware from the releases page
  2. Put your device in bootloader mode (double-tap reset)
  3. Copy the new .uf2 file to the mass storage device
  4. Device automatically reboots with new firmware

How It Works

The Vail Adapter presents itself to your computer as two separate devices:

  • USB HID Keyboard: Sends Ctrl+Left (dit) and Ctrl+Right (dah) key presses
  • USB MIDI Device: Sends MIDI control messages for advanced features

Keyer Modes

The adapter implements all nine keyer modes from Vail:

  1. Straight Key
  2. Bug
  3. Single Paddle
  4. Iambic A
  5. Iambic B
  6. Ultimatic
  7. Single Dot
  8. Single Dash
  9. Beacon
  10. Koch

Switch between modes using MIDI control messages or software interface.

Troubleshooting

Common Issues

Device Not Recognized

  • Try a different USB cable
  • Ensure the device is powered (LED should light up)
  • Re-enter bootloader mode and re-flash firmware
  • Check Windows Device Manager or Mac System Report

Keys Not Working

  • Verify wiring connections
  • Check pin assignments in your firmware configuration
  • Test with different software (try both Vail and a text editor)
  • Ensure software is configured to accept the correct key codes

Timing Issues

  • Enable sidetone for immediate audio feedback
  • Switch to built-in keyer mode for zero-latency operation
  • Adjust keyer speed in software or via MIDI controls

Capacitive Touch Problems

  • Ensure good electrical contact with conductive pads
  • Avoid touching multiple pads simultaneously
  • Keep pads dry and clean
  • Adjust sensitivity in firmware if needed

Advanced Features

Special Function Activation

The Vail Adapter includes several hidden features activated by special key combinations:

Buzzer Disable (5-Second Dit Hold)

Hold the dit paddle for 5 seconds to toggle the sidetone buzzer on/off. Useful for silent operation.

Radio Mode (10 Rapid Dah Presses)

On Advanced PCB models, press the dah paddle 10 times rapidly to toggle radio keying mode for direct transceiver control.

Settings Persistence

The adapter stores your preferences in EEPROM memory:

  • Keyer type selection
  • Dit duration (WPM setting)
  • Sidetone frequency
  • Custom configurations

MIDI Control Interface

Advanced users can send MIDI messages to control adapter settings:

  • Control Change 1: Set dit duration (WPM)
  • Control Change 2: Set sidetone frequency
  • Program Change: Select keyer mode (0-9)

Custom Firmware Development

  • Customize timing parameters

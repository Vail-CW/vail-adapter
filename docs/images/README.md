# Button Hat Modification Images

This directory contains images referenced by `button-hat-mod.html`.

## Images Included

### 1. `diagram.png`
- **Description:** QT Py SAMD21 pinout diagram
- **Purpose:** Reference diagram showing the location of pins A3 and A8
- **Content:** Arduino pin diagram for QT Py SAMD21

### 2. `1.jpg`
- **Description:** Example of completed button hat modification (view 1)
- **Purpose:** Show the jumper wire routing from A3 pad to A8 pin
- **Content:** Photo of actual solder job showing the modification

### 3. `2.jpg`
- **Description:** Example of completed button hat modification (view 2)
- **Purpose:** Alternative angle showing wire routing and solder quality
- **Content:** Photo of actual solder job from different angle

## Image Guidelines

- **Format:** JPEG (.jpg) recommended for photos
- **Resolution:** At least 1200px on the longest side for detail
- **Focus:** Clear, well-lit photos showing the modification details
- **Annotations:** Optional but helpful - add arrows or labels to highlight key areas

## Fallback Behavior

If images are not present, the guide page will display placeholder text explaining what the image should show. Users can still complete the modification using the text instructions.

## Adding Images

1. Copy your images to this directory
2. Ensure filenames match exactly: `qtpy-pinout.jpg`, `button-hat-mod-example1.jpg`, `button-hat-mod-example2.jpg`
3. Commit and push to update the GitHub Pages site
